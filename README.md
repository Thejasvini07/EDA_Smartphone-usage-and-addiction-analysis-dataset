Smartphone Usage & Addiction Analysis

📌 Overview
This project performs Exploratory Data Analysis (EDA) on a smartphone usage dataset to understand user behavior and identify patterns related to smartphone addiction. It also includes basic feature engineering and a logistic regression model.

🔍 Work Done
Cleaned the dataset (handled missing values, removed duplicates, dropped unnecessary columns)
Performed outlier detection using boxplots and IQR method
Conducted univariate analysis using histograms and countplots
Conducted bivariate analysis with respect to addiction label
Created new features like usage category, app usage intensity, and heavy user
Encoded categorical data and applied feature scaling
Built a Logistic Regression model for prediction

📊 Key Insights
Higher screen time is linked to higher addiction
Heavy users are more likely to be addicted
App usage intensity affects addiction behavior

🛠️ Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
